### API's
There are 32 CURD apis based on 8 tables

### USER APIS
#### Create User (`PSOT` ./api/user)
##### request body
```json
{
    "name":"Sahil",
    "username":"sahil_rrey4",
    "email":"email@yahooj.com",
    "location":"any"
}
```

##### response body
SUCCESS
```json
{
    "status": true,
    "message": "user created successfully!"
}
```
